## Open Data Sources

This application uses real open data from the following sources:

### 1. Data Produksi Tanaman Pangan Jawa Timur
- **Source**: Pemerintah Provinsi Jawa Timur
- **URL**: https://data.jatimprov.go.id/dataset/data-produksi-tanaman-pangan-kabupaten-kota-jawa-timur
- **License**: Open Data (CC BY-SA 4.0)
- **Description**: Production data for rice, corn, and soybeans across East Java districts
- **Usage**: Agricultural production statistics and analytics

### 2. Data Irigasi dan Jaringan Irigasi Jawa Timur
- **Source**: Dinas PUPR Provinsi Jawa Timur
- **URL**: https://data.jatimprov.go.id/dataset/data-irigasi-dan-jaringan-irigasi-jawa-timur
- **License**: Open Government Data (OGD Indonesia)
- **Description**: Irrigation network infrastructure data including condition and maintenance
- **Usage**: Irrigation system monitoring and planning

### 3. Data Curah Hujan BMKG Jawa Timur
- **Source**: BMKG (Badan Meteorologi, Klimatologi, dan Geofisika)
- **URL**: https://data.bmkg.go.id/prakiraan/cuaca/curah-hujan
- **License**: BMKG Open Data (Non-commercial, attribution required)
- **Attribution**: "Data dari BMKG (Badan Meteorologi, Klimatologi, dan Geofisika)"
- **Description**: Daily rainfall and weather data from BMKG monitoring stations
- **Usage**: Weather monitoring and climate analysis for agriculture

### 4. Data Potensi Energi Terbarukan Jawa Timur
- **Source**: Dinas ESDM Provinsi Jawa Timur
- **URL**: https://esdm.jatimprov.go.id/data/potensi-energi-terbarukan
- **License**: Open Government Data (OGD Indonesia)
- **Description**: Renewable energy potential data (solar, wind, biomass, micro-hydro)
- **Usage**: Energy planning and sustainability analysis for IoT systems

### 5. Data Kualitas Air Sungai Surabaya
- **Source**: Dinas Lingkungan Hidup Provinsi Jawa Timur
- **URL**: https://dlh.jatimprov.go.id/data/kualitas-air-sungai-surabaya
- **License**: Open Access (attribution required)
- **Attribution**: "Data dari Dinas Lingkungan Hidup Provinsi Jawa Timur"
- **Description**: Water quality monitoring data for Surabaya River system
- **Usage**: Environmental monitoring and water quality analysis

## Data Attribution Display

All data sourced from the above providers is used in compliance with their respective licenses and attribution requirements. When this application is deployed in production, proper attribution will be displayed in the UI where the data is presented.
